git排除大文件

find . -size +1G | cat >> .gitignore

大于号：将一条命令执行结果(标准输出, 或者错误输出, 本来都要打印到屏幕上面的)重定向其它输出设备(文件, 打开文件操作符, 或打印机等等)

小于号：命令默认从键盘获得的输入, 改成从文件, 或者其它打开文件以及设备输入

>> 是追加内容

> 是覆盖原有内容
